# Installation
The project can be easily installed and runned.

To install:

Backend


# Database

For database you do not need to configure anything, you just need to fill you database with dummy data ( for testing purposes ) 

```
npm run seed
```

# Run

Backend

```
npm run server
```

Frontend

Go to frontend directory

```
cd client
```

Start the server

```
npm run start
```
